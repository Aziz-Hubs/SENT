# Vibe Coding Coordination Workflow

// turbo-all

1. **Check:** Before starting, read `/PROJECT_STATE.md`.
2. **Claim:** Add your ID, Task, and Target Files to the 'Active Agent Registry'.
3. **Verify:** Ensure no other agent is working on the same files or conflicting modules.
4. **Code:** Perform the task with high velocity.
5. **Sync:** If an architectural decision is made, update the 'Architectural Guardrails' section.
6. **Clear:** Upon completion, move your entry to 'Coordination Logs' and clear your claim.
